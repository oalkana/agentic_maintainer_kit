# Pull Request Review Packet

## Mode

Read-only review unless the maintainer explicitly authorizes patches.

## Scope

- Repository:
- Branch or PR:
- Files or diff range:
- Areas out of scope:

## Agent instructions

- Prioritize bugs, regressions, security issues, missing tests, and release risk.
- Do not rewrite style unless it affects correctness or maintainability.
- Do not commit, push, or merge.
- Cite exact file paths and line numbers.

## Evidence chain

- Diff inspected:
- Commands run:
- Tests reviewed:
- Reports reviewed:

## Required output

- Findings ordered by severity.
- Open questions.
- Test gaps.
- Short summary.
- Status: `REVIEW_READY`, `PATCH_RECOMMENDED`, or `BLOCKED_NEEDS_MAINTAINER`.

## No-authority statement

This review does not approve merge, release, deployment, or production use.
